# What is docker?
---
A tool that lets you run your apps in containers
---

# Why?
---
```notes
1. security
    if you just installed mysql and node on the same server, and your database was hacked, they have access to everything
    in docker, if your database is hacked, doesn't mean they have access to the node app
2. distribution
    run anywhere
    works on my machine
3. Cleanup
```
